Implemented Classes:
    Rune = defines a single runic path rune, including its identity, realm/category, display data, and effect metadata.
    Runes = acts as the central registry/access point for all defined rune instances and lookup by ResourceLocation.
    RunicRuneData = stores a player’s unlocked runes and currently selected runes per major realm, and handles reading/writing that data to NBT.
    SyncRunes = syncs unlocked and selected rune data from the server to the client for UI/display purposes.
    ClientRunicData = holds the client-side cached copy of unlocked and selected runes so the GUI can render runic information without querying server-only state.
    RealmRuneSelection = represents one major realm’s selected rune list for packet syncing.
    RunicRealm = defines the different major rune categories/realms, such as Flesh, Soul, Spark, and Void.
    RuneEffectType = categorizes what kind of effect a rune provides, such as a stat boost or a passive-skill-style effect.
    RunicPathHelper = provides the core utility logic for rune data access, selection checks, slot handling, and refreshing rune-based passive skills.
    RunicScalingHelper = contains shared helper methods for scaling rune effects based on Runic path progression, especially major realm level.
    RunicTechnique = extends the generic technique system with Runic-specific behavior like cultivation skill setup, passive rune refreshes, max realm limits, and per-realm rune slot counts.
    RunicCultivationSkill = handles the active cultivation behavior for the Runic path.
    RunicCultivationSkillData = stores any persistent/runtime data needed by the Runic cultivation skill.
    RunicArmorSkill = applies the Armor rune’s passive defensive bonuses, mainly direct armor and armor toughness scaling.
    RunicStrengthSkill = applies the Strength rune’s passive bonus by increasing the custom Strength stat.
    RunicVitalitySkill = applies the Vitality rune’s passive bonus by increasing the custom Vitality stat, which then feeds derived survivability attributes through stat scaling.
    RuneItem = represents rune unlock items that grant a rune, optionally auto-select it if a slot is available, save the updated rune data, refresh passives, and sync the result to the client.
    RunicCultivationBoostSkill = a passive Soul rune skill that enables a global increase to cultivation gain while active.
    CultivationModifierHelper = a central utility that calculates and applies all active multipliers affecting cultivation gain across every path.
